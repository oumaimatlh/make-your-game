# 👻 Pac-Man DOM

A browser-based **Pac-Man** game built entirely with **HTML, CSS, and Vanilla JavaScript**. Guide Pac-Man through the maze, collect every dot, avoid the ghosts, and clear the level while aiming for the highest score.

## ✨ Features

- Classic Pac-Man gameplay
- Smooth keyboard controls
- 60 FPS game loop using `requestAnimationFrame()`
- Pause and resume functionality
- Restart option
- Live score tracking
- Lives counter
- Game timer
- Ghost AI and collision detection
- Responsive interface

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/KaoutarHammaoui/PacMan.git
```

or 


```bash
git clone https://github.com/oumaimatlh/make-your-game.git
```

Run the project using a local server (recommended):

```bash
python3 -m http.server
```

or

```bash
npx serve
```

Then open the provided local URL in your browser and click on **maze.html**.
